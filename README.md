# IATA Events Newsletter

This repository contains a responsive HTML email newsletter designed as a portfolio sample based on an events communication use case.

The project follows email-safe development practices, focusing on compatibility across email clients, structured content, and clear information delivery.

## Live Preview
👉 https://jimenezjaik-web.github.io/IATA-Events-Newsletter/

## Key Features
- Table-based layout for maximum email-client compatibility
- Responsive structure adapted for desktop and mobile viewing
- Clear event-focused content hierarchy
- Externally hosted images for reliable email delivery
- Readable typography and structured sections

## Technical Notes
- Built specifically for HTML email (not standard web layouts)
- Inline and email-safe CSS practices applied
- Designed to handle limited CSS support across clients
- Graceful degradation when images are disabled

## Use Case
This project represents an events newsletter, commonly used by organizations to communicate upcoming conferences, announcements, or industry activities. It demonstrates practical HTML email development skills for real-world scenarios.

---

📌 **Note:** Rendering may vary across email clients due to differences in CSS support and client-specific behaviors.
